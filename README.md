# REST API ДЛЯ СИСТЕМЫ УЧЕТА НОМЕРОВ В ГОСТИНИЦЕ
## Описание предметной области 

Проект представляет собой бэкенд-сервис для автоматизации учета номеров в гостинице и управления связанными данными:

- Управление номерами: создание, редактирование, удаление и получение информации о номерах
- Аутентификация и авторизация: использование JWT-токенов для защиты эндпоинтов
- Управление бронированиями: хранение и обработка данных о заселении и проживании клиентов
- Работа с пользователями: учет сотрудников, ролей и учетных данных
- Валидация данных: проверка корректности вводимых данных на уровне API
- Обработка запросов: поддержка стандартных CRUD-операций и корректная обработка ошибок

  ## Стек технологий
- ЯП: .NET 10 и C# 14
- Фреймворк: ASP.NET Core
- ORM: Entity Framework Core
- Безопасность: JWT + ASP.NET Core Identity
- Валидация: FluentValidation
- JSON: System.Text.Json
- UI: Swagger
  
  ## Запуск проекта
Требования: 
- .NET 10 SDK
- PostgreSQL 15+
